## 34.4.1 关于二级虚拟目录

通常我们的站点都是部署在网站的根目录下的，但是有些时候，我们可能将网站根目录下的目录作为二级站点或子站点，这个时候，就会出现 `404` 错误了。

这个时候就需要进行一些特殊配置了。

### 34.4.1.1 配置 `AppSettings`

我们只需要配置 `AppSettings` 即可：

```json
{
  "AppSettings": {
    "VirtualPath": "/虚拟目录"
  }
}
```
